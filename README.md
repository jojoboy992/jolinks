## 📄 Page Structure

The page consists of three main sections:

- **Loading Modal** — A full-page light-themed loader that displays on page load, featuring a spinning SVG ring around the JoNotify logo, animated bouncing dots, and a fade-out transition once the page is ready.
- **Card** — The main content card containing the JoNotify logo/avatar, title, tagline, and three navigation pills (Play Store, WhatsApp, Website).
- **Footer** — A simple credit line: *Built with ❤ — JoNotify*.

---

## ✨ Features

- **Light-themed loading modal** with a purple accent spinner, pulsing subtitle, and bouncing dot indicators
- **Smooth fade-out** — loader disappears 300ms after the page `load` event fires
- **Pill-style links** with icons, labels, subtitles, and arrow indicators
- **Fully responsive** card layout centered on the page
- **No dependencies** — pure HTML, CSS, and vanilla JavaScript

## 🚀 Deployment

The page is hosted on [Netlify](https://netlify.com). To deploy your own copy:

1. Clone or download the project files
2. Open `index.html` in your browser to preview locally
3. Push to a GitHub repo and connect it to Netlify, **or**
4. Drag and drop the project folder directly into the Netlify dashboard
